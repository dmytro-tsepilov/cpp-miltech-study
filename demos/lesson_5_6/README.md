# Demo для заняття 5.6

Малі ROS 2 приклади для фінального заняття блоку 5.

## Приклади

```text
demos/lesson_5_6/
  01_msg_array/
    robot_ws/src/msg_array_demo/
```

## Що де шукати

- `01_msg_array` - власний `.msg` з масивом іншого `.msg`, publisher, який
  заповнює масив через `push_back`, і subscriber, який проходиться по масиву.
